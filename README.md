## アプリ名
Github Releases download counter

## 簡単な説明
任意のGithubリポジトリのリリースから各Assetsのダウンロード数を取得する

## 詳しい説明
https://github.com/ユーザー名/リポジトリ名/releases  
か  
https://github.com/ユーザー名/リポジトリ名/releases/tag/タグ名  
のどちらかを指定して下さい。  
  
あるいは取得用のapiでも構いません。
